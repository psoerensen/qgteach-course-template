# AGENTS.md

## Project purpose

This repository contains Quarto- and R-based teaching material for a course.

## Working guidelines

- Keep teaching material clear, accurate, and reproducible.
- Preserve valid Quarto, Markdown, YAML, and R syntax.
- Prefer small, reviewable edits.
- Do not overwrite unrelated collaborator changes.
- Do not modify raw data or add dependencies without approval.
- Do not render the complete website unless explicitly asked.
- Keep generated website output in `docs/`.
- Check `git status --short` before editing and report checks after editing.
